## Overview

WoodStack is a high-performance digital agency portfolio designed to showcase modern frontend engineering, conversion-oriented UX design, and strategic digital presentation.

The project demonstrates the intersection between:

- Product-focused frontend development
- Performance optimization
- Technical SEO
- Brand storytelling
- Conversion-driven interface design

It was built as both a professional portfolio and a proof-of-concept for scalable digital agency landing page architecture.

---

## Engineering Focus

WoodStack explores key frontend engineering principles:

- Performance-first rendering
- Modular component architecture
- SEO optimization
- Responsive-first layout systems
- Conversion-oriented UX structure
- Maintainable UI composition

---

## Tech Stack

| Layer | Technology | Purpose |
|------|------------|---------|
| **Frontend Core** | React 18 + Vite | Fast development & optimized builds |
| **Routing** | React Router DOM | Seamless SPA navigation |
| **Styling** | CSS Modules + TailwindCSS | Scalable styling system |
| **Deployment** | Vercel | CI/CD & preview deployments |
| **SEO** | React Helmet + meta tags | Search optimization |

## Local Development

### Clone Repository

```bash
git clone https://github.com/MartinBernardoBonilla/woodstack-portfolio.git
cd woodstack-portfolio
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```
